i created a tool to visualize 3d math functions and linear transformations. users can enter a math function, and it will graph with coloring based on x, y, and z axis. users can also apply linear transformations such as shear, rotate, and scale. the program was written in C++ using SFML.

to learn more, check it out on youtube or github.
